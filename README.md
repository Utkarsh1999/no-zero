# NoZero: Minimalist Habit Consistency Tracker

NoZero is a high-impact, minimalist habit tracking application built with Compose Multiplatform. It focuses on the psychological reinforcement of consistency rather than just streaks or intensity.

## Project Overview
- **Goal**: Help users build good habits and avoid bad ones through identity-based reinforcement.
- **Philosophy**: "No Zero" - prioritizing recovery and resilience over perfect streaks to avoid toxic guilt.
- **Stack**: Kotlin, Compose Multiplatform (Android + iOS), Clean Architecture, MVVM, SQLDelight/Room.
